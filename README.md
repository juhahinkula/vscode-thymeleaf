## Features

Thymeleaf snippets for vscode

Snippets for the most common thymeleaf features. Works with .html files.

| Trigger  | Content |
| -------: | ------- |
| `thtxt`   | th:text="${}" |
| `thfld`   | th:field="*{}" |
| `theach`   |"th:each=" : ${}" |
| `theachli`  | Thymeleaf each with list |
| `thobj`  | th:object="${}" |
| `thref`  | th:href="@{}" |
| `thform`  | Thymeleaf form |
| `thif`   | th:if="${}" |
| `thswitch`  | th:switch="${}" |
| `thcase`  | th:case="" |
| `thval`  | th:value="" |
| `thins`  | th:insert="" |
| `threp`  | th:replace="" |
| `thfr`  | th:fragment |
| `thwith`  | th:with="" |
| `thstyle`  | th:style="" |
| `thtmpl`  | HTML thymleaf template |
