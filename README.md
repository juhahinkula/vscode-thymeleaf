## Features

Thymeleaf snippets for vscode

Snippets for the most common thymeleaf features

| Trigger  | Content |
| -------: | ------- |
| `thtxt`   | th:text="${}" |
| `theach`   |"th:each=" : ${}" |
| `theachli`  | <li th:each=" : ${}"></li> |
| `thobj`  | th:object="${}" |
| `thref`  | th:href="@{}" |
| `thform`  | <form th:action="@{}"></form> |
| `thif`   | th:if="@{}" |
| `thswitch`  | th:switch="${}" |
| `thcase`  | th:case="" |
| `thval`  | th:value="" |
| `thtmpl`  | HTML thymleaf template |
